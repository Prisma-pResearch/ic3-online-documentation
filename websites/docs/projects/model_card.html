Model Card for MySurgeryRisk

Model Details

Overview

MySurgeryRisk predicts the risk for eight main postoperative complications (acute kidney injury, sepsis, venous thromboembolism, prolonged intensive care unit stay >48 hours, prolonged mechanical ventilation >48 hours, wound, neurologic, and cardiovascular complications) and death up to 30/90 days after surgery.

Considerations

Intended Users

  • Medical professionals
  • ML researchers

Use Cases

  • Input: electronic health records data including demographics, admission infomration, scheduled surgery information, comorbidity, laboratory measurements and medications within one year prior to surgery
  • Output: Risk of eight postoperative complications and death (value range [0, 1])
  • Machine learning model: Random Forest
  • Usage: Before surgery, MySurgeryRisk will estimate the risk of complications and inform the doctor.

Limitations

  • The generalizability of MySurgeryRisk model is limited. The input features such as zip code and doctor id depend on the dataset.

Ethical Considerations

  • Risk: Manual selection of input features could create selection bias
    Mitigation Strategy: Automate the selection process
  • Risk: Imbalanced dataset could create bias
    Mitigation Strategy: Upsampling/downsampling strategy
