## Datasets Workflow
IC<sup>3</sup> data core takes the responsibility to **manage** the data collected from UF-IDR(Integrated Data Registry). All data will be processed and stored with below brief workflow.<br />

```mermaid
%%{init: {'theme':'base'}}%%
flowchart LR
A[Raw data] -- IDR data pipeline --> B[EHR/Text data] & G[Image data] & I[Waveform data]
B -- Data Cleaning Pipeline --> C[Cleaned EHR data]
B -- Variables generation Pipeline --> D[Generated variables]
G & I  -- Data Indexing Pipeline --> H[File metadata]
C & D & H -- OMOP Mapping Pipeline --> F[(databases)]

```
Researchers are recommended to **query the structured databases** and download the sub-cohort for each project. <br />
Please feel free to contact with [IC<sup>3</sup> data core](mailto:ziyuan.guan@ufl.edu,shickelb@ufl.edu,Tezcan.OzrazgatBaslanti@medicine.ufl.edu) about any datasets problems.
